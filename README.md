# AgentHack

A comprehensive agent-based system for managing calendars, weather, emails, and automated tasks using AO (Autonomous Objects) processes.

## Components

- **`agent/`** - Core AO agent processes (Lua)
- **`utilities/`** - Node.js utility scripts for testing and management
- **`relay_monitor/`** - Relay process monitoring system
- **`relay_process/`** - Mock relay implementation
- **`crontroller/`** - Cron job management system

## Usage

This software is designed to work with AO processes and requires proper configuration of environment variables. See the individual component directories for specific usage instructions.
